# @vben/turbo-run

`turbo-run` is a CLI utility that lets you choose packages interactively and run one script across them in parallel.

## Features

- interactive package selection
- monorepo-aware script discovery
- accurate target filtering

## Install

```bash
pnpm add -D @vben/turbo-run
```

## Usage

```bash
turbo-run [script]
```

Example:

```bash
turbo-run dev
```
