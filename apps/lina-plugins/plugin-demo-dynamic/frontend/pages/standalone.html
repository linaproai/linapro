This page is served directly by plugin-demo-dynamic as a hosted static asset rather than through the Vben frontend framework. It verifies that a dynamic plugin can render inside the host workspace and can also open a full standalone experience in a new window.

What This Page Verifies

  • Hosted Asset Delivery The page file is served from the plugin-hosted asset directory instead of a host source route.
  • Independent Presentation The standalone page does not reuse Vben components or layout, but it still keeps a polished and readable presentation.
  • End-to-end Navigation Users can jump here from the host workspace and verify the flow from embedded mount to standalone access.

Current Plugin Shape

plugin-demo-dynamic is a dynamic plugin sample used to demonstrate directory conventions, Wasm artifact delivery, hosted assets, and the host mount protocol.

Design Goal

This page intentionally avoids the default admin shell style and demonstrates how a plugin can deliver its own static visual language while remaining professional and readable.
