# 登录框默认布局改为居中

## 背景

当前登录页在无宿主覆盖时默认使用 `panel-right`（右对齐）。产品期望将登录框默认布局调整为居中（`panel-center`），管理员仍可通过系统参数 `sys.auth.loginPanelLayout` 在左/中/右之间切换。

## 范围

- 更新登录页展示规范：默认布局由右对齐改为居中
- 同步宿主种子默认值、后端公共前端默认回退、前端 preferences 与公共配置回退
- 更新相关单元测试与 E2E 默认布局断言

## 非目标

- 不改变左/中/右三种布局的渲染能力
- 不移除登录页工具栏中的布局切换器
- 不改变其他登录展示参数（标题、描述、副标题、入口开关等）
