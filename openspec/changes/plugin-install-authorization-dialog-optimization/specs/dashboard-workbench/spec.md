# Dashboard Workbench

## ADDED Requirements

### Requirement: 工作台首页必须展示与 LinaPro 管理后台一致的真实入口
系统 SHALL 将默认工作台首页中的快捷入口、项目卡片和运维待办收敛为与 LinaPro 管理后台一致的真实导航与运维语义，不得继续保留无法访问的模板演示链接。

#### Scenario: 点击工作台快捷入口

- **WHEN** 管理员在工作台首页点击快捷入口
- **THEN** 宿主跳转到可访问的 LinaPro 内部页面或明确的外部官方资源
- **AND** 不再出现 `/demos/...` 这类模板占位路由

#### Scenario: 浏览工作台首页内容

- **WHEN** 管理员打开工作台首页
- **THEN** 页面展示的项目、动态和待办内容围绕核心宿主服务、默认管理工作台、插件扩展能力和研发协作流程组织
- **AND** 页面文案保持与当前项目定位一致，不再沿用无关的模板示例叙事

### Requirement: 分析页必须提供清晰的指标语义与时间范围切换
系统 SHALL 在默认分析页中提供可切换的时间范围视角，并为关键指标区块和图表区块使用明确且不重复的标题，帮助管理员快速理解宿主工作区运行概况。

#### Scenario: 切换分析时间范围

- **WHEN** 管理员在分析页切换预设时间范围
- **THEN** 页面使用对应时间范围的数据刷新概览指标、趋势摘要和洞察信息
- **AND** 当前选中的时间范围在界面上保持明显的激活态

#### Scenario: 浏览分析图表分区

- **WHEN** 管理员查看分析页的图表卡片
- **THEN** 每个卡片标题都应准确表达其展示内容
- **AND** 页面不再出现两个含义不同却同名的图表标题
