## MODIFIED Requirements

### Requirement: 登录后返回用户信息

系统 SHALL 在用户登录成功后返回当前用户可访问的首页路径，避免前端进入无权限或不存在的页面。

#### Scenario: 用户存在可访问菜单
- **WHEN** 用户登录成功且拥有至少一个可访问的菜单路由
- **THEN** 系统返回的 `homePath` 必须指向该用户首个可访问的菜单路由
- **AND** 该路径必须能够被前端动态路由正常解析

#### Scenario: 用户不存在可访问菜单
- **WHEN** 用户登录成功但没有任何可访问的业务菜单
- **THEN** 系统返回的 `homePath` 必须回退到一个已注册且可访问的安全页面
- **AND** 系统不能将用户重定向到 404 页面作为默认登录结果
