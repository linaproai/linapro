## ADDED Requirements

### Requirement: 插件管理页支持查看插件详情

系统 SHALL 在插件管理页为每条插件记录提供“详情”入口，并通过弹窗展示该插件的完整治理信息。

#### Scenario: 管理员从列表查看插件详情

- **WHEN** 管理员在插件管理页点击某条插件记录的“详情”按钮
- **THEN** 系统打开详情弹窗
- **AND** 弹窗展示该插件的名称、标识、类型、版本、描述、接入状态、当前状态、授权要求、授权状态、安装时间与更新时间

#### Scenario: 插件声明宿主服务权限

- **WHEN** 当前插件存在宿主服务申请清单或授权快照
- **THEN** 详情弹窗展示对应的服务类型、方法集合与资源边界
- **AND** 管理员无需离开列表页即可查看治理边界

#### Scenario: 插件未声明宿主服务权限

- **WHEN** 当前插件不存在宿主服务申请清单和授权快照
- **THEN** 详情弹窗显示无宿主服务治理信息的提示
- **AND** 基础信息仍然正常展示
