## ADDED Requirements

### Requirement: 插件安装弹窗支持安装并启用快捷操作

系统 SHALL 在插件安装弹窗中同时提供“仅安装”和“安装并启用”两类治理动作，让管理员在完成插件信息核对后按需直接完成快捷启用，而无需先安装再回到列表二次操作。

#### Scenario: 同时具备安装与启用权限时展示快捷动作
- **WHEN** 管理员打开一个未安装插件的安装弹窗，且当前账号同时具备 `plugin:install` 与 `plugin:enable` 权限
- **THEN** 弹窗展示“安装并启用”快捷动作
- **AND** 弹窗仍保留“仅安装”动作，供管理员显式选择只完成安装

#### Scenario: 仅具备安装权限时隐藏快捷动作
- **WHEN** 管理员打开一个未安装插件的安装弹窗，但当前账号仅具备 `plugin:install` 权限而不具备 `plugin:enable`
- **THEN** 弹窗只展示“仅安装”动作
- **AND** 界面 MUST NOT 展示“安装并启用”按钮，避免暗示用户可以执行超出权限的治理动作

#### Scenario: 组合动作在第二步失败时展示真实状态
- **WHEN** 管理员在安装弹窗中选择“安装并启用”，且安装成功但启用失败
- **THEN** 插件管理页刷新后必须展示该插件处于“已安装、未启用”状态
- **AND** 界面明确提示管理员安装已完成但启用未成功，可后续再次手动启用
